1. The Home page has the functionalites of Navbar on top body and footer subsequently
2. In the body we have cards that show the name, Cuisines, Rating of a Particular restaurant.
3. These are fetched by the public API of swiggy.
4. There is a search box on top of Cards where a user can search for a Particular restaurant. 
5. The functionality of Search box has been implemented using Hooks - UseEffect and UseState
6. we also added shimmer effect to show a loading gif during the time of fetching the API.
7. we added 404 Page to show when a Search result is not found 
8. We are constantly Updating the UI and functionalites.
